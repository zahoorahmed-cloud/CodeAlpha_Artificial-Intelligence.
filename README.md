## 🛠️ Project Portfolio

### 🌍 Task 1: Language Translation Tool
* [cite_start]**Description:** A smart translation utility providing seamless text conversion across global languages[cite: 21, 22].
* [cite_start]**Tech Stack:** Python, Google Translate / Microsoft Translator API, UI Framework[cite: 23].
* [cite_start]**Core Features:** Dynamic source/target selection, real-time API request processing, clean UI layout[cite: 22, 24, 25].

### 🤖 Task 2: NLP Chatbot for FAQs
* [cite_start]**Description:** An intelligent automated assistant capable of matching user inquiries against a pre-compiled knowledge base[cite: 27, 28].
* [cite_start]**Tech Stack:** Python, NLTK / SpaCy, Scikit-learn (Cosine Similarity)[cite: 29, 30].
* [cite_start]**Core Features:** Text tokenization, lemmatization, semantic intent matching, and responsive chat logic[cite: 29, 30, 31].

### 🎵 Task 3: Algorithmic Music Generation AI
* [cite_start]**Description:** A generative deep learning model trained on symbolic MIDI data to compose original musical patterns[cite: 35, 36].
* [cite_start]**Tech Stack:** Python, TensorFlow / Keras, `music21`, LSTMs / RNNs[cite: 37, 38].
* [cite_start]**Core Features:** MIDI sequence preprocessing, deep neural network training, automated sequence-to-audio export[cite: 37, 39, 41].

### 👁️ Task 4: Object Detection and Tracking
* [cite_start]**Description:** A high-performance real-time computer vision system that detects specific objects and tracks them continuously across frames using deep appearance metrics[cite: 42, 43, 46].
* [cite_start]**Tech Stack:** Python, OpenCV, Ultralytics YOLOv8, Deep SORT (`deep-sort-realtime`)[cite: 43, 44, 46].
* [cite_start]**Core Features:** Live webcam stream handling, sub-millisecond object classification bounding boxes, robust track ID persistence during overlaps, and dynamic on-screen tracking analytics[cite: 43, 45, 47].
